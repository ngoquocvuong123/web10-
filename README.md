# Editor 4: CSS làm nổi bật thành phần URL

## Mô tả
Trang HTML hiển thị một URL được chia thành 3 phần: **protocol**, **domain**, **path**,
mỗi phần được gán một class riêng và tô màu khác nhau bằng CSS để dễ phân biệt.

## Cấu trúc project
```
.
├── index.html
├── style.css
└── README.md
```

## Cách chạy project
1. Clone hoặc tải repo về máy
2. Mở file `index.html` bằng trình duyệt để xem trang

## GitHub Pages
1. Vào repo trên GitHub → **Settings** → **Pages**
2. Ở mục **Source**, chọn nhánh `main` và thư mục `/ (root)`
3. Đợi vài phút và truy cập link dạng:
   `https://<tên-người-dùng>.github.io/<tên-repo>/`
